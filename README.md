# Homebrew Tap

Personal Homebrew tap for @cipherchabon's projects.

## Installation

```bash
brew tap cipherchabon/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [clinbox](https://github.com/cipherchabon/clinbox) | Terminal-first email client with AI-powered triage |
| [svg2font](https://github.com/cipherchabon/svg2font) | Convert SVG icons to TTF icon font |

## Usage

```bash
# Install a formula
brew install cipherchabon/tap/<formula>

# Or after tapping
brew install <formula>
```
